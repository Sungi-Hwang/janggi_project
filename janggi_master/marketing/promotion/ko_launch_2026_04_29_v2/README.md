# 장기 한수 공유용 홍보 이미지 v2

기존 v1 이미지가 여백이 많고 홍보물 느낌이 약해 보여서, 공유용 카드 형태로 다시 제작한 버전입니다.

## 변경점

- 보드/앱 화면을 블러 배경으로 사용해 빈 공간을 줄임
- 문제 번호, 난이도, CTA 버튼을 추가
- 긴 URL을 제거하고 `Google Play에서 “장기 한수” 검색`으로 정리
- 커뮤니티, 유튜브 커뮤니티, 카카오 오픈채팅, 블로그 대표 이미지에 맞는 `1080x1080` 규격

## 파일

- `01_today_problem_beginner.png`: 오늘의 장기 문제
- `02_viewer_challenge.png`: 시청자 도전 문제
- `03_replay_analysis_problem.png`: 기보 복기 문제
- `04_ai_hint_challenge.png`: AI 추천수 맞히기
- `05_community_problem_board.png`: 장기 문제 공유소
- `preview_contact_sheet.jpg`: 전체 미리보기

## 권장 사용

첫 홍보글에는 `01_today_problem_beginner.png`를 사용합니다.

본문 문구는 이전 폴더의 `post_templates.md`를 그대로 쓰되, 이미지 파일만 이 v2 폴더의 파일로 교체합니다.
