# 장기 한수 한국 시장 무예산 홍보 세트

Date: 2026-04-29

## 목표

앱 자체를 광고하기보다 `오늘의 장기 문제` 콘텐츠를 먼저 보여주고, 앱은 문제를 바로 풀어보는 도구로 연결한다.

1차 KPI:

- 1주: Play Store 방문자 200+, 설치 50+, 리뷰 5개
- 2주: Play Store 방문자 500+, 설치 100+, 리뷰 10개

## 실행 순서

1. `01_today_problem_beginner.png`와 `post_templates.md`의 첫 홍보글을 사용해 첫 게시물을 올린다.
2. 같은 날 지인/테스터 10명에게 `review_request.md` 문구로 리뷰를 요청한다.
3. 2~4일차에 하루 1개씩 다른 이미지와 문구를 올린다.
4. 유튜버 후속 메일은 최초 발송 후 3~5일 뒤에만 보낸다.
5. 매일 `channel_tracker.csv`에 게시 위치, 반응, 설치 추정 효과를 기록한다.

## 운영 원칙

- 커뮤니티 규칙을 먼저 확인한다.
- 같은 곳에 하루 여러 번 올리지 않는다.
- 앱 홍보글처럼 쓰지 말고 문제풀이 글처럼 쓴다.
- 댓글이 달리면 정답 공개보다 “앱에서 풀어볼 수 있다”는 흐름을 유지한다.
- 부정 반응이 있으면 바로 홍보를 멈추고 피드백으로 기록한다.
