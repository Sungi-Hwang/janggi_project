# 1.2.0+13 release notes

## ko-KR

- 문제 공유소를 추가했습니다. 다른 사용자가 공유한 문제를 최신순/좋아요순으로 둘러보고 가져올 수 있습니다.
- 문제 공유소 검색을 추가했습니다. 제목과 작성자명으로 문제를 빠르게 찾을 수 있습니다.
- 문제풀이 화면을 1수, 2수, 3수, 가져온 문제 탭으로 정리했습니다.
- 공유 코드와 커뮤니티에서 가져온 문제를 별도 보관함에 저장하도록 개선했습니다.
- 내가 만든 문제 편집기의 모바일 조작감을 개선했습니다. 기물을 선택하고 보드를 탭해서 배치할 수 있습니다.
- 광고 설정을 실제 배포용 AdMob ID로 갱신했습니다.
- 여러 안정성 문제와 내부 데이터 정리를 개선했습니다.
