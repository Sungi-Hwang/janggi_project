# 스토어 영상 구성안

## 결론

- Google Play 스토어 영상은 보통 `YouTube 링크`로 연결합니다.
- 그래서 먼저 `가로형 16:9 영상` 하나 만드는 게 가장 안전합니다.
- 베타 모집용으로도 그대로 재활용할 수 있게 `15초` 안팎으로 짧게 가는 걸 추천합니다.

## 권장 길이

- `15초~20초`

## 권장 비율

- `16:9 가로`
- 예: `1920x1080`

## 장면 구성

### 1. 0:00 ~ 0:03

- 메인 화면
- 자막: `AI 훈수 + 묘수풀이 + 이어하기 분석`

### 2. 0:03 ~ 0:08

- 퍼즐 플레이 화면
- 자막: `정답이 납득되는 퍼즐 위주로 다시 정리`

### 3. 0:08 ~ 0:12

- 퍼즐 완료 표시 화면
- 자막: `푼 문제는 바로 체크`

### 4. 0:12 ~ 0:15

- 이어하기 또는 AI 대국 화면
- 자막: `혼자 장기 실력을 늘리는 훈련 앱`
- 마지막 자막: `베타테스터 모집 중`

## 자막 대본

AI 훈수 + 묘수풀이 + 이어하기 분석

정답이 납득되는 퍼즐 위주로 다시 정리

푼 문제는 바로 체크

혼자 장기 실력을 늘리는 훈련 앱

베타테스터 모집 중
